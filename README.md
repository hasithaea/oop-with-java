# Object Oriented Programming with Java

A collection of Java programs, exercises, and practical implementations created while learning **Object Oriented Programming (OOP)** concepts as part of the **Bachelor of Computer Science** degree at the **University of Ruhuna, Sri Lanka**.

The repository focuses on understanding and applying core object oriented principles using Java through hands-on examples and practical exercises.

## Topics Covered

### Fundamentals

* Variables and Data Types
* Operators and Expressions
* Control Structures
* Methods and Functions
* Arrays and Strings

### Object-Oriented Programming

* Classes and Objects
* Constructors
* Encapsulation
* Information Hiding
* Access Modifiers
* Inheritance
* Method Overriding
* Polymorphism
* Abstraction
* Interfaces
* Composition and Delegation

### Advanced Concepts

* Exception Handling
* File Handling
* Collections Framework
* Generics
* Enumerations
* Packages
* Wrapper Classes
* Lambda Expressions

### UML and Design

* Class Diagrams
* Object Relationships
* Basic Object-Oriented Design Concepts

## Technologies Used

* Java
* IntelliJ IDEA
* JDK 25
* Git & GitHub

## Running the Programs

Compile:

```bash
javac Main.java
```

Run:

```bash
java Main
```

If using packages:

```bash
javac package_name/Main.java
java package_name.Main
```

## Learning Objectives

* Understand object oriented programming principles.
* Develop reusable and maintainable software components.
* Apply abstraction and encapsulation to real-world problems.
* Design programs using inheritance and polymorphism.
* Build a strong foundation for advanced Java frameworks and enterprise development.
